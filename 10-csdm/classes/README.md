# Classes

One markdown note per CI class or table that has surfaced in real engagement work. File naming convention:

- **Technical table name, lowercase, underscores**: `cmdb_ci_service_business.md`, `u_sp_ebond_config.md`
- This matches grep-ability against logs/blueprints that use the same identifier.

Use [../_template-class.md](../_template-class.md) as the starting template.

Each note should be linked from at least one [domain index](../README.md) so the graph stays connected.
